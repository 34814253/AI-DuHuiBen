## 1. 当前代码已引用（建议优先补齐）

以下为 **路径已在代码里写好**，放入同名文件即可生效；缺失时会有 **计时模拟 / 渐变占位 / Web Audio 蜂鸣** 等降级。

### 1.1 音频（MP3，PRD 建议约 128kbps）

| 文件路径（相对 `public/`） | 用途 | 与 `durationSec` 关系 |
| -------------------------- | ---- | --------------------- |
| `mock-audio/book1-page1.mp3` | 《快乐的光脚丫先生》第 1 页朗读 / 其它书短 Demo 第 1 页 | `bookPages.ts` 中约 **5s**（短 Demo 为 4s） |
| `mock-audio/book1-page2.mp3` | 第 2 页 | 约 **4s** |
| `mock-audio/book1-page3.mp3` | 第 3 页 | **6s**（通用 4s） |
| `mock-audio/book1-page4.mp3` | 第 4 页 | 约 **5s** |
| `mock-audio/book1-page5.mp3` | 第 5 页 | 约 **4s** |
| `mock-audio/bgm.mp3` | P4 朗读页背景音乐（循环） | 任意时长循环曲 |

**说明**：实际朗读时长若与 `durationSec` 不一致，将以 **音频真实播放进度** 为准（`ended` 触发翻页）；若加载失败则退回 **定时逻辑**（与表中秒数对齐）。

### 1.2 图片（JPG / PNG，PRD §9）

当前 **`bookPages.ts` 未配置 `imageUrl`**，P4 使用渐变 + emoji 占位。若要上图：

| 建议路径 | 用途 |
| -------- | ---- |
| `mock-assets/covers/book1.png` | 扫码绘本封面（代码已写 `coverUrl`） |
| `mock-assets/pages/book1-page1.png` … `book1-page5.png` | P4 分页插图（代码已写 `imageUrl`） |

**最小集**：至少 **5 张分页图 + 1 张封面** 即可贯穿扫码→书架→朗读主路径（需配合少量代码填写路径）。

## 4. 格式与体积建议（衔接 PRD §9）

| 类型 | 建议 |
| ---- | ---- |
| **朗读 / BGM** | **MP3**，约 **128kbps**；音量可做轻度归一，避免页间忽大忽小 |
| **插图 / 封面** | **JPG 或 PNG**；如需兼容与体积可另存 **WebP**，但 **不能替代** JPG/PNG 时需保留双格式策略 |
| **命名** | 与 `bookPages.ts` 中路径 **完全一致**（大小写敏感） |
